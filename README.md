# Botanical Bites

**Description:**  
A modern, plant-themed restaurant website designed to showcase the menu, story, and dining experience. Features a clean UI, visually rich elements, and intuitive navigation, allowing customers to explore dishes and engage with the brand.

**Features:**  
- Menu display with categories  
- About and story section  
- Responsive design for mobile and desktop  
- Contact details and reservation form  

**Tech Stack:**  
- HTML, CSS, JavaScript  
- Optional backend: PHP or Node.js (if reservations or dynamic content)  

**Installation:**  
1. Clone the repository: `git clone <repo-url>`  
2. Open `index.html` in a browser  

**Usage:**  
- Browse the menu, read about the restaurant, and view contact info  
- Fully responsive on all devices
